# Robocon
[VB6] Posicionamento de um robô de 6 eixos (Positioning a 6-axis robot)

Este é um pequeno software para determiniar o posicionamento do cabeçote no espaço de um manipulador robótico do tipo DELTA, ao movimentar cada motor/eixo.

-------------------------------------------------------------
[VB6] Positioning a 6-axis robot (Positioning a 6-axis robot)

This is a small software to determine the position of the head in the space of a robotic manipulator of the DELTA type, when moving each motor/axis.
